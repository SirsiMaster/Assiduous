# Infra (Deployment & Infrastructure)

This directory holds deployment scripts and infrastructure configuration for Assiduous Realty.

- Cloud Run deployment specs for Go backend services
- (Optional) Terraform or gcloud-based provisioning scripts
- SQL schema files for Cloud SQL
- Any shared CI/CD helper scripts that are not specific to frontend or backend

All commands and deployment flows referenced here must also be documented in `docs/DEPLOYMENT_GUIDE.md` and `ARCHITECTURE_DESIGN.md` for canonical reference.
