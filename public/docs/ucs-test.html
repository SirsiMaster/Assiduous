How It Works

The UCS build system:

  1. Found this file: public/docs/ucs-test.template.html
  2. Detected the @component:sidebar directive
  3. Loaded public/components/sidebar.html
  4. Calculated paths: Since we're at docs/, ../ = ../
  5. Replaced all ../ tokens with ../
  6. Injected the processed sidebar HTML
  7. Wrote output: public/docs/ucs-test.html (this file!)
