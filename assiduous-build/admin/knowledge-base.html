Essential Development Commands

Development Server

python -m http.server 8080 - Start local development server

python serve.py - Alternative server (if serve.py exists)

open http://localhost:8080/ - Access application

Git Operations

./scripts/hooks/install.sh - Install git hooks (required after cloning)

git tag -l | sort -V - View available tags

git push origin --tags - Push tags to remote

Version Management

./scripts/rollback/create_tag.sh 0.2.0 "Release notes" - Create release tag

./scripts/changelog/update_changelog.sh [version] - Update changelog

./scripts/rollback/rollback_to_tag.sh v0.1.0 - Rollback to previous version

Key Services & Components

  • firebaseservice.js - Core database operations with encryption
  • auth.js - Authentication and session management
  • crm.js - Client relationship management
  • encryptionservice.js - Field-level encryption layer

Firebase Collections

  • users - Encrypted user profiles (email, phone, SSN)
  • properties - Listings with investment scoring
  • transactions - Deal pipeline management
  • messages - Communication logs
  • notifications - Real-time alerts

Critical Architecture Rules

  • NEVER store sensitive data in GitHub (use Firebase)
  • ALWAYS encrypt sensitive fields before Firestore storage
  • USE Firebase for all backend operations (no GitHub Actions for DB)
  • MAINTAIN unified client role (no separate buyer/seller paths)
  • PRIORITIZE micro-flipping features (70% of dev effort)

Quick Start Guide

For Executives

  1. Open the Executive Dashboard for high-level overview
  2. Review key metrics and performance indicators
  3. Access detailed reports via backend modules
  4. Export data for board presentations

For Agents

  1. Access Agent Portal to view your performance
  2. Manage your property listings
  3. Track client interactions in CRM
  4. Monitor deal progress in transactions

For Property Management

  1. Navigate to Properties Module
  2. Add new listings with images and details
  3. Update property status and pricing
  4. Export inventory reports

For Market Analysis

  1. Visit Market Analytics
  2. Review current market metrics
  3. Analyze neighborhood performance
  4. Generate market reports

Pro Tip

All data is automatically saved as you work. Use the export features in each module to generate reports for external use.
